# Python电商数据分析学习平台

一个纯前端的Python在线学习平台，使用Pyodide在浏览器中运行Python代码。

## 功能特点

- 📚 完整课程体系（Python基础 + 电商数据分析实战）
- 💻 10个固定电商练习任务
- 📝 测评考试系统（5道单选 + 5道代码题）
- 🔥 内置Python编辑器，支持Pandas和DataFrame展示
- 🎨 蓝白风格，简约界面
